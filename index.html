Watch OUT!

  • High Concept: Move mario with the mouse to avoid the bombettes as long as you can!
  • Player Experience Goals: The player should be involved in extreme focus and anticipation. The game will hopefully be fun enough to introduce a competitive desire to beat their own and others time record.
  • Genre: A single sentence that places the game within a game genre or a hybrid of genres.
  • Game play: Mario will follow the mouse. Move Mario around, to avoid the bombettes. The bombettes will speed up the longer you last, last as long as you can!
  • Prototype goal: This game demonstrates the core functionality of controlling a sprite with mouse, and detecting collisions with enemy sprites.
  • Features: This is a really simple game that tests concentration and reactions. The enemies always start in random places, so each play is different.
  • Setting/Premise: This game doesn't really have a setting or premise. I just used Mario because they were the first pictures I found when I googled ".png". The pictures can be anything, including rectangles with the same experience. I would say this game is abstract.
  • Story: Bombs want to kill Mario, Mario doesnt want to die!
  • Target Audience: Ages 9 - 99.
  • Play time: If you last more than 30 seconds, you're decent. If you last more than 40 seconds, you're really good!
  • Strategies: Look past the screen, don't focus on a single sprite. Look past the screen so that you can see the movements of everything on the screen.
